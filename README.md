## How to Install

### Method 1: Load from URL (Recommended)
Once hosted on GitHub Pages:
1. Open Google AI Edge Gallery
2. Agent Skills → Skills → (+) → Load skill from URL
3. Enter: `https://YOUR_USERNAME.github.io/edge-gallery-skills/SKILL_NAME`

### Method 2: Import Local Skill
1. Copy the skill folder to your phone
2. Open Google AI Edge Gallery
3. Agent Skills → Skills → (+) → Import local skill
4. Select the skill folder
